cart = [];  
  
  
function addToCart(product, price) { 
cart.push({ 	product, 	price 	}); 
alert(product + " added to cart!"); 
displayCart();  
}  
  
  
function displayCart() { let cartList =  
document.getElementById("cartItems"); cartList.innerHTML = "";  
let total = 0;  
  
  
cart.forEach(item 	=> 	{ 	let 	li 	= 
document.createElement("li"); li.textContent = item.product + " - ₹" + item.price; 
cartList.appendChild(li); total += item.price;  
});  
  
  
document.getElementById("totalPrice").textContent = "Total: ₹" + total;  
}  



PWA create karne ke liye aapko koi special software install nahi karna hai, bas ye 5 steps follow karein:

1. Folder Structure Setup
Ek folder banayein (e.g., ShopEasy) aur usme ye structure rakhein:

index.html (Main UI)

manifest.json (App Config)

js/script.js (Cart Logic)

js/sw.js (Service Worker - isko abhi khali file rakhein)

images/ (Isme icon.png aur smartphone.jpg rakhein)

2. Files Save Karein
Jo code aapne upar diya hai, usko copy karke sahi files mein save karein.

Tip: Agar aapke paas images nahi hain, toh internet se koi bhi random photo download karke same naam se save kar dein.

3. Service Worker (Zaroori)
js/sw.js file banayein aur usme ye basic code dal dein (taaki PWA installable ban jaye):

JavaScript
self.addEventListener('fetch', (event) => {
  // Filhal ye empty hai, par PWA ke liye iska hona zaroori hai
});
4. Local Server Start Karein
Browser security ki wajah se PWA direct file open (file://) karne par nahi chalta.

VS Code: Live Server extension install karke "Go Live" par click karein.

Browser: http://127.0.0.1:5500/index.html jaisa link open hoga.

5. Install & Run
Browser mein link open hone ke baad, URL bar ke right side mein "Install" icon (desktop par) ya "Add to Home Screen" (mobile par) ka option aayega.

Install karne ke baad, ye app ki tarah aapke desktop/mobile menu mein dikhega.
