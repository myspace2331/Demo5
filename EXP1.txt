To install and configure Flutter Environment.

Steps to install flutter:

sudo apt update && sudo apt upgrade -y

sudo apt install curl git unzip xz-utils zip libglu1-mesa -y

cd Downloads

curl -O https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_latest-stable.tar.xz

tar xf flutter_linux_latest-stable.tar.xz

sudo mv flutter /opt/

echo 'export PATH="$PATH:/opt/flutter/bin"' >> ~/.bashrc
source ~/.bashrc

flutter doctor

sudo snap install android-studio --classic

Then:

Open Android Studio
Install SDK
Install Emulator
Accept licenses:
flutter doctor --android-licenses

flutter doctor
